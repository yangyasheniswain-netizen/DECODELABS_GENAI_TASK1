# AI Chatbot using Google Gemini API

## 📌 Project Overview
This project is a simple AI Chatbot built using Python and Google Gemini API. The chatbot can interact with users, answer questions, and maintain a conversation through the terminal.

## 🚀 Features
- Interactive chatbot
- Google Gemini API integration
- Real-time responses
- User-friendly terminal interface
- Exit command support

## 🛠️ Technologies Used
- Python
- Google Gemini API
- VS Code

## 📂 Project Files
- chatbot.py
- test_models.py

## ▶️ How to Run
1. Install required libraries
2. Add your Gemini API Key
3. Run:
